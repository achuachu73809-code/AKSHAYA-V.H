✨ Features

* Average marks calculation
* Grade assignment
* Pass/Fail status
* Subject-wise average analysis
* Student performance comparison
* Performance dashboard using charts
* Data visualization

  🛠 Technologies Used

* Python 3
* Pandas
* NumPy
* Matplotlib

* Calculate each student’s average marks.
* Assign grades based on average scores.
* Determine pass/fail status.

* Compare student and subject performance.
* Visualize results using charts.
